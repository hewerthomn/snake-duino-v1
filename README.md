# Snake Duino v1

Arduino Snake game

Inspirated in [Snake v1.0][1], Ouarrak Ayoub

## Components

- Nokia 5110 LCD attached to pins 7, 6, 5, 4, 3
- Active Buzzer attached to pin 8
- Push-buttons attached to pins A2, A3, A4, A5
