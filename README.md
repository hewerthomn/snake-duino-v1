##SnakeDuino v1
--------
Arduino Snake game

####Components
Nokia 5110 LCD attached to pins 7, 6, 5, 4, 3 <br>
Active Buzzer attached to pin 8 <br>
Push-buttons attached to pins A2, A3, A4, A5<br>

Inspirated in Snake v1.0, Ouarrak Ayoub<br>
